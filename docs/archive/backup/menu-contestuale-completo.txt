================================================================================
ISTRUZIONI PER RIMOZIONE
================================================================================

Per rimuovere una voce dal menu contestuale:

1. Aprire regedit (Windows + R, digitare "regedit")

2. Per CARTELLE, navigare a:
   HKEY_CLASSES_ROOT\Directory\shell\[NOME_VOCE]

3. Per BACKGROUND CARTELLE, navigare a:
   HKEY_CLASSES_ROOT\Directory\Background\shell\[NOME_VOCE]

4. Click destro sulla chiave → Elimina

5. Riavviare explorer.exe o fare logout/login

BACKUP CONSIGLIATO: Prima di eliminare, click destro sulla chiave → Esporta
(salvare come file .reg per eventuale ripristino)
